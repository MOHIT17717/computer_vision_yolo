# Computer Vision YOLO Projects (computer_vision_yolo)

This repository contains computer vision projects focusing on real-time object detection using YOLO models. 

## Projects included:

### [Garbage Overflow Detection](garbage_overflow_detection/)
A smart city project utilizing YOLOv8 and OpenCV for real-time garbage bin detection, fill-level estimation, and overflow alerting. Includes a web dashboard using Flask for monitoring.

For full details, installation instructions, and usage, please refer to the `garbage_overflow_detection/README.md`.
